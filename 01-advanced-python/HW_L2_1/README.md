This repository contains the solutions for the Python Basics homework assignments in the Data Science Bootcamp.

## 📌 صورت سوالات و تمرین‌ها (Assignments Description)

### 1️⃣ تمرین اول: محاسبه شاخص توده بدنی (Practice 1: BMI Calculator)
* **فایل کد:** `Practice1.py`
* **توضیحات (FA):** برنامه‌ای بنویسید که نام، سن، قد (به متر) و وزن (به کیلوگرم) کاربر را از ورودی دریافت کرده، شاخص BMI را محاسبه کند و خروجی را با دو رقم اعشار نمایش دهد.
* **Description (EN):** Write a program that takes the user's name, age, height (in meters), and weight (in kilograms) as input, calculates their BMI, and displays the result formatted to 2 decimal places.

---

### 2️⃣ تمرین دوم: تحقیق درباره متد `split` (Practice 2: Deep Dive into `split()`)
* **فایل گزارش:** `Practice2.pdf`
* **توضیحات (FA):** با استفاده از تابع `help()` در پایتون، درباره متد `split` تحقیق کنید. نحوه کارکرد، آرگومان‌های ورودی (`sep` و `maxsplit`) را همراه با مثال‌های مختلف به صورت یک فایل گزارش ارائه دهید.
* **Description (EN):** Investigate the `split()` method using Python's built-in `help()` function. Explain its functionality, input arguments (`sep` and `maxsplit`), and provide practical examples in a comprehensive report.

---

### 3️⃣ تمرین سوم: شمارش تکرار یک مقدار در لیست (Practice 3: Count Element in List)
* **فایل کد:** `Practice3.py`
* **توضیحات (FA):** برنامه‌ای بنویسید که ۴ مقدار را از کاربر دریافت کرده و در یک لیست ذخیره کند. سپس یک مقدار دیگر نیز دریافت کرده و تعداد دفعات تکرار آن مقدار را در لیست مشخص کند.
* **Description (EN):** Write a program that accepts 4 values from the user and stores them in a list. Then, take one more value and count how many times that specific value appears in the list.

---

### 4️⃣ تمرین چهارم: پردازش و مهندسی متن (Practice 4: String Processing & Manipulation)
* **فایل کد:** `Practice4.py`
* **توضیحات (FA):** برنامه‌ای بنویسید که یک متن انگلیسی را دریافت کرده و مراحل زیر را روی آن انجام داده و در هر مرحله خروجی را چاپ کند:
  1. حذف نقطه‌ها و ویرگول‌ها.
  2. جداسازی کلمات و ذخیره در یک لیست.
  3. مرتب‌سازی کلمات بر اساس حروف الفبا (حساس نبودن به حروف بزرگ و کوچک).
  4. ساخت یک رشته جدید با کلمات مرتب شده (با فاصله میان آن‌ها).
  5. معکوس کردن نیمه دوم جمله جدید و چاپ آن.
* **Description (EN):** Write a program that receives a text string in English and performs the following operations, displaying the output at each step:
  1. Remove periods (.) and commas (,).
  2. Split the text into a list of words.
  3. Sort the words alphabetically (case-insensitive).
  4. Join the sorted list into a new sentence.
  5. Reverse the second half of the newly generated sentence and print it.
